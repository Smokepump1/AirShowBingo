AIRSHOW PEOPLE BINGO - VERSION 1

How to run it:
1. Open the airshow-bingo-app folder.
2. Double-click index.html.
3. It will open in your web browser.
4. Click New Card to reshuffle.
5. Click any square to mark it.
6. The center FREE square is already marked.

How to edit the item list:
1. Right-click app.js.
2. Open it with VS Code.
3. Find MASTER_ITEMS near the top.
4. Add or edit items inside the quotation marks.
5. Save the file.
6. Refresh the browser.

How to keep it organized:
Put this folder somewhere easy, such as:
Desktop\Airshow Bingo
or
Documents\Projects\Airshow Bingo
