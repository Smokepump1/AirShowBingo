# Airshow People Bingo

A simple 5x5 mobile-friendly bingo card for airshow people watching.

## Files

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## How it works

- Tap a square to mark it.
- Center square is FREE.
- New Card shuffles the list.
- Reset Marks clears the board.
- Works through GitHub Pages.
